### 符号共享

- EXPORT_SYMBOL(name)：导出的符号对所有内核模块可见（最常用）。
- EXPORT_SYMBOL_GPL(name)：导出的符号仅对遵循 GPL 协议的内核模块可见（用于限制闭源模块使用，符合 GPL 许可证要求）。

### 示例

- 模块 B 中定义并导出函数：

  ```c
  void my_shared_func(void) {
  	// 实现逻辑
  }
  EXPORT_SYMBOL(my_shared_func);  // 导出符号
  ```

- 模块 A 中即可直接调用该函数（需包含对应声明）：

  ```c
  extern void my_shared_func(void);  // 声明外部符号
  void func_in_moduleA(void) {
  	my_shared_func();  // 调用模块B导出的函数
  }
  ```
