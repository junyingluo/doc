### 编译

- 使用 CMake 构建系统

  - git clone --branch llvmorg-15.0.0 https://github.com/llvm/llvm-project.git
  - mkdir build
  - cd build
  - cmake -G "Unix Makefiles" -DLLVM_ENABLE_PROJECTS="clang" -DLLVM_TARGETS_TO_BUILD="X86" -DCMAKE_BUILD_TYPE=Release ../llvm
  - make -j8

- 使用 Ninja 构建系统
  - cmake -G "Ninja" -DLLVM_ENABLE_PROJECTS="clang" -DLLVM_TARGETS_TO_BUILD="X86" -DCMAKE_BUILD_TYPE=Release ../llvm
  - ninja
