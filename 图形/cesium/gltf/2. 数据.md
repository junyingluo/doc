### 数据（buffers）

- 外部数据

```json
{
  "buffers": [
    {
      "uri": "model.bin",
      "byteLength": 12345
    }
  ]
}
```

- 内部数据

```json
{
  "buffers": [
    {
      "byteLength": 4508,
      "extras": {
        "_pipeline": {
          "source": new Uint8Array([...]),
        },
      },
    },
  ]
}
```

### 数据视图（bufferViews）

- bufferViews 将 buffers 按类型进行划分

* target 表示该数据是顶点数据
  - 34962 表示 gl.ARRAY_BUFFER，用 drawArrays
  - 34963 表示 gl.ELEMENT_ARRAY_BUFFER，用 drawElements

```json
{
  "bufferViews": [
    {
      "buffer": 0, // 引用第 0 个数据
      "byteOffset": 0, // 起始偏移量
      "byteLength": 1356, // 数据长度
      "target": 34962 // ARRAY_BUFFER
    },
    {
      "buffer": 0,
      "byteOffset": 1356,
      "byteLength": 904,
      "target": 34962
    },
    {
      "buffer": 0,
      "byteOffset": 2260,
      "byteLength": 990,
      "target": 34963
    },
    {
      "buffer": 0,
      "byteOffset": 3252,
      "byteLength": 1255
    }
  ]
}
```
