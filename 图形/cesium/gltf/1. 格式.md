### 总体

```json
{
  "asset": { "version": "2.0", "generator": "Blender" },
  "scenes": [], // 场景数组（包含节点层级）
  "nodes": [], // 节点数组（变换、子节点引用）
  "meshes": [], // 网格数组（几何体数据）
  "materials": [], // 材质数组（PBR 渲染属性）
  "accessors": [], // 访问器数组（类型化数据视图）
  "bufferViews": [], // 缓冲区视图数组（缓冲区切片）
  "buffers": [], // 缓冲区数组（二进制数据容器）
  "textures": [], // 纹理数组（图片引用）
  "images": [], // 图片数组（纹理源）
  "samplers": [], // 采样器数组（纹理过滤/环绕）
  "cameras": [], // 相机数组（视角参数）
  "skins": [], // 蒙皮数组（骨骼动画）
  "animations": [], // 动画数组（关键帧数据）
  "scene": 0 // 默认场景索引
}
```
