### 属性

- frustum
  - 默认透视投影
    - this.frustum = new PerspectiveFrustum();
  - aspectRatio（宽高比）
    - this.frustum.aspectRatio = scene.drawingBufferWidth / scene.drawingBufferHeight
  - fov（60 度）
    - this.frustum.fov = CesiumMath.toRadians(60.0)

* \_projection
  - 默认 GeographicProjection

  ```js
  function GeographicProjection(ellipsoid) {
    this._ellipsoid = ellipsoid ?? Ellipsoid.default;
    this._semimajorAxis = this._ellipsoid.maximumRadius;
    this._oneOverSemimajorAxis = 1.0 / this._semimajorAxis;
  }
  ```

* \_transform 平移矩阵
* \_viewMatrix 视图矩阵
* \_invViewMatrix 视图矩阵的逆矩阵（实际上的视图矩阵）

### 方法

- updateViewMatrix 计算 \_viewMatrix

```js
function updateViewMatrix(camera) {
  Matrix4.computeView(
    camera._position,
    camera._direction,
    camera._up,
    camera._right,
    camera._viewMatrix,
  );
  Matrix4.multiply(
    camera._viewMatrix,
    camera._actualInvTransform,
    camera._viewMatrix,
  );
  Matrix4.inverseTransformation(camera._viewMatrix, camera._invViewMatrix);
}
```
